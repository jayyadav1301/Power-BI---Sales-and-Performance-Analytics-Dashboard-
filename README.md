Awesome Chocolates – Power BI Sales & Performance Dashboard
Project Overview

This project presents an interactive Power BI dashboard built for Awesome Chocolates to analyze sales performance, profitability, customer trends, and product insights.
The dashboard is designed to help business stakeholders make data-driven decisions by converting raw sales data into clear, actionable insights.

It focuses on answering key business questions such as:
How are sales and profits trending over time?
Which products and categories drive the most revenue?
Which regions and customers contribute the most to performance?
Where are the growth opportunities and performance gaps?

Business Objectives:
Track overall Sales, Profit, and Quantity performance
Identify top-performing and underperforming products
Analyze regional and customer-wise performance
Monitor trends over time for better forecasting and planning
Support management decision-making with visual, easy-to-read insights

Key KPIs & Metrics:
Total Sales
Total Profit
Profit Margin
Total Quantity Sold
Sales & Profit Trend (Time Series)
Top / Bottom Products by Sales & Profit
Region-wise and Customer-wise Performance
Category and Product-level Contribution

Tools & Technologies Used:
Power BI – Data modeling, DAX measures, and dashboard development
Excel / CSV (Source Data) – Raw data storage and preprocessing
DAX – For calculated measures, KPIs, and business logic
Power Query – Data cleaning, transformation, and shaping

Key Features of the Dashboard
Interactive slicers for Date, Region, Product, and Category
Dynamic KPIs with automatic recalculation based on filters
Trend analysis using line and area charts
Performance comparison using bar charts, tables, and cards
Clean, business-focused layout for management reporting
Drill-down capability for detailed analysis

File Information:
Awesome Chocolates Dashboard.pbix – Main Power BI dashboard file
Data is modeled using a star-schema style structure (fact & dimension tables)
Measures are created using DAX for accurate and dynamic reporting

How to Use:
Download and open the .pbix file in Power BI Desktop
Refresh the data (if source files are connected)
Use slicers to filter by time period, region, product, or category
Explore different pages/visuals to analyze performance from multiple angles
Use insights for business decisions, presentations, or case studies

Insights You Can Derive:
Which products generate the highest revenue and profit
Which regions or customers are most valuable
Where profit margins are low despite high sales
How sales and profit change over time
Which categories need strategic focus or optimization
